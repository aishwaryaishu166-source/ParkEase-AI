## About the Project
ParkEase AI helps users find and book parking slots easily. It includes a simple admin panel to manage parking stations and bookings.
The system also includes a basic AI feature that suggests parking based on availability.

## Features
- User registration and login
- Search parking stations
- Book parking slots
- View booking history
- Admin dashboard for management
- Basic AI parking suggestion feature

## System Flow
User → Login → Search Parking → Book Slot → Admin Manages Data → AI Suggests Parking

## How it Works
1. User logs in or registers
2. Searches available parking
3. Books a slot
4. Admin manages parking data
5. AI suggests suitable parking options based on availability
